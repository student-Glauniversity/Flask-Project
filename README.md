## 📌 Project Overview

This project is a Flask-based CRUD (Create, Read, Update, Delete) web application built using Python and Flask.

It demonstrates full CRUD functionality with a structured backend and dynamic frontend rendering.

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2 Templates
- SQLite / Database
- Render (Deployment)

---

## 🎯 Features

- ➕ Create new records  
- 📖 View existing records  
- ✏️ Update records  
- ❌ Delete records  
- Dynamic routing  
- Template rendering  
- Deployed on cloud platform
